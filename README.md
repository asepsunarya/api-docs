# Fumadocs + Scalar Starter

Starter dokumentasi API berbasis Next.js, Fumadocs UI, dan Fumadocs OpenAPI/Scalar playground.

## Development

```bash
npm install
npm run dev
```

Buka:

- `/docs` untuk dokumentasi MDX
- `/docs/openapi` untuk API reference interaktif

## OpenAPI

Edit schema di `openapi.json`. Fumadocs OpenAPI akan membangun halaman API dari schema tersebut saat build/dev.
