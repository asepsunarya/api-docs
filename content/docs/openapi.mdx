Gunakan halaman ini sebagai pintasan utama untuk membuka endpoint Socialchat Partner API yang valid.

## Pilih kategori API

- [Partner API](/docs/openapi/partner-api/PartnerController_getPartner) — informasi partner, OAuth, Zapier, dan konfigurasi callback.
- [Channel API](/docs/openapi/channel-api/ChannelController_getChannel) — daftar channel, detail channel, dan konfigurasi channel.
- [Conversation API](/docs/openapi/conversation-controller/ConversationController_getConversation) — daftar percakapan, detail percakapan, dan assignment.
- [Message API](/docs/openapi/message-api/MessageController_searchMessage) — pencarian dan pengelolaan pesan.
- [Webhook / Callback](/docs/openapi/partner-api/PartnerController_updatePartner) — konfigurasi webhook dan callback URL partner.
- [Channel WhatsApp Official API](/docs/openapi/channel-whatsapp-official-api/WhatsAppOfficialController_createAccount) — koneksi WhatsApp Official, template message, dan media.
- [Channel WhatsApp Unofficial API](/docs/openapi/channel-whatsapp-unofficial-api/WhatsAppUnofficialController_createAccount) — koneksi WhatsApp Unofficial.

## Sebelum mencoba endpoint

Pastikan setiap request menyertakan header autentikasi berikut:

```http
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

Jika belum memiliki API key, baca halaman [Authentication](/docs/authentication) terlebih dahulu.
