## Selamat datang

Project ini sudah berisi setup **Fumadocs** untuk halaman dokumentasi dan **Fumadocs OpenAPI** untuk render referensi API interaktif berbasis Scalar.

Mulai dari halaman [OpenAPI](./openapi) untuk melihat endpoint contoh.

## Yang sudah dipasang

- Next.js App Router
- Fumadocs UI + MDX
- Fumadocs OpenAPI
- Search endpoint lokal
- Proxy endpoint untuk playground API
